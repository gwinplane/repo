# AddressManagementSystem

A **.NET (C#) Console Application** for managing addresses.  
Implements basic **CRUD operations** (Create, Read, Update, Delete).

---

## Description

This application allows you to add, view, edit, and delete addresses
through a console menu. The project is created for educational purposes
and demonstrates a structured approach to .NET Console Application development.
